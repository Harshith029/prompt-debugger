# Pattern Library (anthropic pack)

Before/after prompt patterns, indexed by [`index.json`](index.json). Each pattern is a markdown file mapping a rubric dimension and technique(s) to a concrete transformation.

**Status at M0:** the index lists three seed patterns as `status: draft`; their markdown bodies are authored in M1 alongside the technique worked-examples. Pattern bodies must respect the rewrite contract's transformation rules — a pattern may make intent more explicit, never less.
